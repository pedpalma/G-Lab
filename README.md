# G-Lab Website
This repository contains the source code for the G-Lab website, which is built using pure HTML, CSS, and JavaScript and hosted on Vercel.
The website serves as a landing page to showcase internet plans and provide information about the services offered by G-Lab Telecom.

# Objective
The objective of the G-Lab website is to provide a simple and effective way for customers to learn about the internet plans and services offered by G-Lab Telecom. The website aims to be user-friendly, visually appealing, and informative, allowing visitors to easily navigate through the content and find the information they need.

# Features
- Responsive design for optimal viewing on various devices
- Interactive elements to engage users
- Clear and concise information about internet plans and services
- Contact information and links to social media profiles

# Technologies Used
- HTML5 for structuring the content
- CSS3 for styling and layout
- JavaScript for interactivity and dynamic content
- Vercel for hosting and deployment

# Deployment
The website is deployed on Vercel, which provides a seamless deployment process and ensures that the site is fast and reliable. Any changes made to the code in this repository will automatically trigger a new deployment, making it easy to keep the website up-to-date with the latest information and features.
